가. Backend 기술 관련 안내



텀 프로젝트에서는 Backend 기술을 반드시 활용해야 합니다. 다음 중 한 가지 방식 또는 병행하여 구현이 가능합니다



(1) 서버에 저장된 데이터 활용

서버에 저장된 데이터를 웹 애플리케이션에서 요청하여 불러오고, 이를 가공해 사용자에게 출력하는 방식



(2) 외부 API 연동

외부에서 제공하는 공개 API 또는 인증 기반 API로부터 데이터를 가져와 가공한 후, 웹 페이지에 동적으로 표시하는 방식



나. 필수 기술 요소



모든 텀 프로젝트에는 다음 기술이 반드시 포함되어야 합니다



  HTML, CSS, 반응형 웹 구현 (필수 요소) 



다. 선택 기술 요소 (LA 주제 중 택 2개 이상)



다음 LA 주제 중 2개 이상의 기술을 프로젝트에 포함시켜야 합니다



(1) DOM을 이용한 데이터 저장 및 검색 기능

(2) 웹 스토리지를 활용한 데이터 저장 및 수정 기능

(3) HTML Graphics 및 Media를 활용한 페이지 구성

(단순히 이미지나 비디오만 삽입하는 것은 인정되지 않음)

(4) PHP 또는 서버 응용프로그램을 이용한 JSON 파일 기반 데이터 저장 및 검색 기능

(5) Session을 이용한 데이터 저장 및 수정 기능

(6) Drag and Drop API + Ajax를 활용한 데이터 저장삭제수정 기능
